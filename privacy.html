Privacy Policy for Stash

Effective Date: December 28, 2025 | Publisher: Power Users Lab

Overview

Stash is a local-first productivity tool. We believe your data is yours. This policy explains what we collect (almost nothing), what we don't collect (everything else), and why.

What Data Does Stash Collect?

None.

Stash does not collect, transmit, or store any personal data on external servers. All snippets, settings, and usage preferences are stored locally on your device using Chrome's storage.local API. This data never leaves your browser.

We do not use analytics, telemetry, cookies, or tracking of any kind.

Why Does Stash Need Certain Permissions?

Permission Why We Need It
storage To save your snippets and settings locally on your device.
activeTab To detect the currently focused input field when inserting a snippet.
scripting To inject saved text into active text fields on any website.
clipboardWrite As a fallback method for inserting text. Not all websites handle direct text injection the same way, so we use clipboard as a backup to ensure Stash works reliably across the web.
Host permissions (all URLs) To allow snippet insertion on any website you visit. We do not read, monitor, or transmit page content.

We request only the permissions necessary for Stash to function. If a feature isn't essential to our core purpose, we don't ask for the permission it would require.

Data Sharing

We do not share, sell, trade, or transfer your data to any third party. There is no data to share because we don't collect any.

Data Security

Since all data is stored locally in your browser and never transmitted, there is no server-side data to protect. Chrome's storage API provides the security boundary.

Your Rights

You have full control over your data:

Access: View your snippets anytime in the Stash interface.

Deletion: Delete individual snippets or clear all data via Settings.

Portability: Export your snippets as a JSON file at any time.

Revocation: Uninstall the extension to remove all stored data.

Children's Privacy

Stash does not collect information from anyone, including children. There is no data collection to speak of.

Changes to This Policy

If we update this policy, we will post the revised version with a new effective date. Continued use of Stash after changes constitutes acceptance of the updated policy.
